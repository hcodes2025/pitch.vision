Crafting Tomorrow's Intelligence

An evolving journey toward personal AI

Mission

To build a future with safe, adaptive, energy-efficient, and personalized AI. These applications combine cutting-edge emerging technologies for real-time learning, memory retention, and modular agency to advance AI built for people.

SNN + Transformers

SpikeGate

Biologically-inspired gating for LLM alignment

A transformer model enhanced with a spiking neural gate to ensure inference-time robustness. Explores energy-efficient safety via neural gating, with contrastive diagnostics across ReLU and Sigmoid baselines.

Agents + Tools

Book Recommender Agent

Modular MCA agent pipeline with real-world hooks

An interactive agent system following emerging agent protocols, capable of planning, retrieving, and generating user-specific book suggestions. Will be reused in SNN-RPi to test modular agent portability.

SNN + SIGNALS

MLP-SNN Rare Event Detector

First MLP-style Spiking Neural Network for EEG Seizure Detection — trained in seconds, deployable on-device.

A minimal MLP-style spiking neural network trained on CHB-MIT data. Demonstrates energy-efficient spike-based learning, with real-world inference code and model included. First such result using MLP-SNN on clinical EEG.

Continual Learning

Personal LLMs (SNN-RPi)

Edge inference using real-time user vectors

Combines spiking learning on keystroke or sensor inputs with downstream LLM biasing. Demonstrates privacy-preserving personalization on-device, tested on MacBook + Raspberry Pi.
